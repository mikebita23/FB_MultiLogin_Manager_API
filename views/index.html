The current functionalities (All transactions are in JSON) :



Authentification

  • To login send a POST request to
    /Auth/login

    Data to send : 

                            {
                            "email": "nidal@gmail.com",
                            "passWord": "nidal321"
                        }

    Return a Bearer Token


Users :

  • User Data format : 
                        {
                        "firstName":"Jhon",
                        "lastName":"Parker",
                        "email":"Hunter_Bauch40@gmail.com",
                        "phoneNumber":"0762612642",
                        "passWord":"123456",
                        "role":null,
                        "forfaitId":null
                    }
  • You can fetch Users with GET request to
    /users/all
    or one user to
    /users/'id'

  • For creating a User send a User in POST request to
    /users/add

  • To Update a User send a User in POST request to
    /users/edit
    ( You can send only the new attributes )

  • For Delete send a GET request to
    /users/remove/'id'
    ( the id shold be in the link )


Messages :

  • Messages Data format : 
                        {
                        "Object": "Obj",
                        "Content": "Message Content",
                    }
  • You can fetch Messages with GET request to
    /Msg/all
    or one Msg to
    /Msg/'id'

  • For adding a Message send a Msg in POST request to
    /Msg/add

  • For Deleting send a GET request to
    /Msg/del/'id'
    ( the id shold be in the link )


Proxy :

  • Proxy Data format : 
                    {
                    url: 'http://lum-customer-{lumUser}-zone-{lumZone}:{password}@zproxy.lum-superproxy.io:{lumPort}'
                }
  • You can fetch Proxy with GET request to
    /proxy/get


How to Download :

  • You can fetch All versions of the app by sending a Get request to
    /Dwn/get

  • Once you chosen your version send a GET request to
    /Dwn/getLink/'name'
    to receive a download link
