Inital instalation

after Clonning the project and creating database execute these commands :

            -- npm install -g --save bcrypt sequelize-cli
            -- npm install
            -- sequelize db:merge
            -- sequelize db:seed:all
            -- npm start

The current functionalities (All transactions are in JSON) :


Authentification

  • To login send a POST request to
    /Auth/login

    Data to send : 

                            {
                            "email": "nidal@gmail.com",
                            "passWord": "nidal321"
                        }

    Return a Bearer Token


Users :

  • User Data format : 
                        {
                        "firstName":"Jhon",
                        "lastName":"Parker",
                        "email":"Hunter_Bauch40@gmail.com",
                        "phoneNumber":"0762612642",
                        "passWord":"123456",
                        "role":null,
                        "forfaitId":null
                    }
  • You can fetch Users with GET request to
    /users/all
    or one user to
    /users/'id'

  • For creating a User send a User in POST request to
    /users/add

  • To Update a User send a User in PATCH request to
    /users/edit/'id'
    ( the id shold be in the link )

  • For Delete send a GET request to
    /users/remove/'id'
    ( the id shold be in the link )


Messages :

  • Messages Data format : 
                        {
                        "Object": "Obj",
                        "Content": "Message Content",
                    }
  • You can fetch Messages with GET request to
    /Msg/all
    or one Msg to
    /Msg/'id'

  • For adding a Message send a Msg in POST request to
    /Msg/add

  • For Deleting send a GET request to
    /Msg/del/'id'
    ( the id shold be in the link )


Forfaits :

  • Forfaits Data format : 
                        {
                        "nom": "nom",
                        "prix": "prix",
                        "description": "description",
                    }
  • You can fetch Forfait with GET request to
    /forf/all
    or one Forfait to
    /forf/'id'

  • For adding a Forfait send a forfait in POST request to
    /forf/add

  • For Deleting send a GET request to
    /forf/del/'id'
    ( the id shold be in the link )

  • For Edit send a GET request to
    /forf/'id'/edit
    ( the id shold be in the link )
